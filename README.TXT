
AzureBlobBackup - command line utility for backing up files to the Azure Blob Storage.
--------------------------------------------------------------------------------------

Please see the reference at http://lundin.info/azureblobbackup.aspx
or simply use the --help command switch. 

Bin folder contains the utility and necessary assemblies, copy this folder
to a suitable location, there is no installer.

Src folder contains the Visual Studio solution for the utility.
